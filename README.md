Introduction

CRFA public webapp. It is Next.js application. Read more here: Next.README.md

```
yarn install
```

# Dev server
```
yarn dev
```

# Prod build
```
yarn build
yarn start
```

# Docker prod build
```
docker build . -t crfa-public-webapp
```

# Docker prod start
```
docker run -p 3000:3000 crfa-public-webapp
```
